# Система автоматизации сервисного обслуживания для ремонтной компании

Выпускная квалификационная работа: веб-приложение для автоматизации сервисного
обслуживания ремонтной компании (ремонт компьютеров и оргтехники) и пояснительная
записка к нему.

Приложение обеспечивает онлайн-приём заявок на ремонт, ведение заказов с полным
жизненным циклом (от приёмки до выдачи), учёт услуг и запчастей, расчёт стоимости,
согласование с клиентом, журнал работ, отзывы, уведомления и сводную аналитику.

## Технологический стек

- **TypeScript**, среда выполнения **Node.js 20+**
- **Next.js 16** (App Router) — серверные компоненты и серверные действия
- **React 19**, **Tailwind CSS 4**
- **Prisma 7** (ORM) + **SQLite** (драйвер better-sqlite3)
- Аутентификация: cookie-сессии с подписью HMAC, хеширование паролей scrypt

## Структура репозитория

```
repair-service/            — веб-приложение (исходный код)
diagrams/                  — диаграммы (use-case, архитектура, ER)
screenshots/               — экранные формы приложения
docgen/                    — генератор пояснительной записки (docx)
shoot.js, diagshoot.js     — скрипты съёмки скриншотов и диаграмм
Пояснительная_записка.docx — пояснительная записка (ГОСТ)
```

## Запуск приложения

Требуется установленный **Node.js 20+**.

```bash
# 1. Перейти в каталог приложения
cd repair-service

# 2. Установить зависимости
npm install

# 3. Создать файл окружения из примера
cp .env.example .env

# 4. Создать базу данных и применить схему
npx prisma db push

# 5. Наполнить демонстрационными данными
npm run seed

# 6. Запустить сервер разработки
npm run dev
```

Приложение откроется по адресу **http://localhost:3000** (если порт занят — 3001).

Продакшен-сборка:

```bash
npm run build && npm run start
```

## Возможности по ролям

- **Гость** — просмотр главной страницы, каталога услуг и цен, регистрация и вход.
- **Клиент** — подача заявок на ремонт, отслеживание статуса и истории заказа,
  согласование стоимости, отзывы, уведомления, профиль.
- **Мастер** — работа с назначенными заказами: смена статуса, диагностика, учёт услуг
  и запчастей с автоматическим расчётом стоимости, журнал работ.
- **Администратор** — управление пользователями и ролями, справочниками (услуги,
  запчасти, категории), всеми заказами и назначением мастеров, сводная статистика.

## Пояснительная записка

Файл `Пояснительная_записка.docx` оформлен по ГОСТ. Содержит реферат, введение, три
главы (анализ предметной области, проектирование, реализация и тестирование),
заключение, список источников и четыре приложения (техническое задание, ER-диаграмма,
листинги, экранные формы).
