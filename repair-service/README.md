# КомпСервис — система автоматизации сервисного обслуживания

Веб-приложение для автоматизации сервисного обслуживания ремонтной компании
(ремонт компьютеров и оргтехники). Выпускная квалификационная работа.

Приложение обеспечивает онлайн-приём заявок на ремонт, ведение заказов с полным
жизненным циклом (от приёмки до выдачи), учёт услуг и запчастей, расчёт стоимости,
согласование с клиентом, ведение журнала работ, отзывы, уведомления и сводную
аналитику для администратора.

## Технологический стек

- **Язык:** TypeScript
- **Среда выполнения:** Node.js 20+
- **Фреймворк:** Next.js 16 (App Router) — серверные компоненты и серверные действия
- **Интерфейс:** React 19, Tailwind CSS 4
- **ORM:** Prisma 7
- **СУБД:** SQLite (драйвер better-sqlite3)
- **Аутентификация:** подписанные cookie-сессии (HMAC), хеширование паролей scrypt

## Роли пользователей

- **Администратор** — управление пользователями, справочниками (услуги, запчасти,
  категории), всеми заказами, назначение мастеров, сводная статистика.
- **Мастер** — работа с назначенными заказами: смена статуса, диагностика, учёт услуг
  и запчастей, журнал работ.
- **Клиент** — подача заявок, отслеживание статуса, согласование стоимости, отзывы.

## Запуск

```bash
# 1. Установка зависимостей
npm install

# 2. Создание базы данных и применение схемы
npx prisma db push

# 3. Наполнение демонстрационными данными
npm run seed

# 4. Запуск в режиме разработки
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000 (если порт занят — на 3001).

Продакшен-сборка:

```bash
npm run build
npm run start
```

## Структура проекта

```
prisma/
  schema.prisma        — модель данных (11 сущностей)
  seed.ts              — демонстрационные данные
src/
  app/
    (public)/          — публичные страницы (главная, услуги)
    (auth)/            — вход и регистрация
    (panel)/           — личный кабинет (клиент, мастер, администратор)
  actions/             — серверные действия (заказы, работа мастера, администрирование)
  components/          — компоненты интерфейса
  lib/                 — подключение к БД, аутентификация, вспомогательные функции
```
